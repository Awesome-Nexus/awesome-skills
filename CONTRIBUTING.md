# Contributing to Awesome Skills

Thank you for your interest in contributing! This document provides guidelines for adding resources to the SKILL framework.

## The Skills Framework

Ensure your contribution fits into one of the five categories:

| Category | Covers | Examples |
|----------|--------|----------|
| **S - Systems** | Foundation models, LLMs, multimodal AI | GPT-4, Llama, Claude |
| **K - Kits** | Development tools, frameworks, SDKs | LangChain, Vector DBs |
| **I - Intelligence** | AI agents, multi-agent systems | CrewAI, AutoGen |
| **L - Learning** | Educational resources | Courses, papers, tutorials |
| **L - Links** | MCP servers, APIs, integrations | MCP servers, plugins |

## How to Contribute

1. **Fork the repository**
2. **Add your resource** to the appropriate category file
3. **Follow the format**: Use the existing table format
4. **Submit a pull request** with a clear description

### Format Template

```markdown
| [Name](https://link) | Provider | Feature | Best For |
```
