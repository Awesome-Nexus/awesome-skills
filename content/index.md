# Awesome Skills

A comprehensive framework for building AI-powered applications with Model Context Protocol.

## Features

- **Systems** - Core infrastructure and frameworks
- **Kits** - Development tools and SDKs
- **Intelligence** - AI/ML resources
- **Learning** - Educational materials
- **Links** - Curated resources
- **Security** - Security tools and practices

## Getting Started

```bash
npm install
npm run dev
```
