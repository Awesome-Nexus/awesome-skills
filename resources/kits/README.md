## LLM Frameworks

| Framework | Language | Focus | Best For |
|-----------|----------|-------|----------|
| [LangChain](https://langchain.com) | Python/JS | Chains, RAG | Complex workflows |
| [LlamaIndex](https://www.llamaindex.ai) | Python/TS | Data ingestion | RAG applications |
| [Haystack](https://haystack.deepset.ai) | Python | Search/NLP | Document search |
| [Semantic Kernel](https://github.com/microsoft/semantic-kernel) | .NET/Py/TS | Microsoft integration | Enterprise apps |
| [LiteLLM](https://github.com/BerriAI/litellm) | Python | Universal API | Multi-provider apps |
| [Instructor](https://github.com/jxnl/instructor) | Python | Structured outputs | Data extraction |
| [Outlines](https://github.com/dottxt-ai/outlines) | Python | Structured generation | Constrained outputs |
| [DSPy](https://github.com/stanfordnlp/dspy) | Python | Prompt optimization | Advanced prompting |
